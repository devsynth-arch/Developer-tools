# Open-Source Developer Tools

A collection of simple, useful tools for developers to enhance productivity and contribute to open-source projects.

## Tools Included

### 1. Markdown Link Checker (`link_checker.py`)
Checks for broken links in Markdown files.
- Usage: `python link_checker.py <markdown_file>`
- Useful for maintaining documentation quality.

### 2. README Generator (`readme_generator.py`)
Generates a basic README.md template for new projects.
- Usage: `python readme_generator.py <project_name>`
- Helps kickstart project documentation.

### 3. Code Line Counter (`line_counter.py`)
Counts lines of code in Python files within a directory, excluding comments and blanks.
- Usage: `python line_counter.py <directory>`
- Great for project statistics and contributions.

## Installation

Clone this repo and ensure you have Python 3 installed.

```bash
git clone https://github.com/yourusername/dev-tools.git
cd dev-tools
```
