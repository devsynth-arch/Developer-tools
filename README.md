# Open-Source Developer Tools

A collection of simple, useful tools for developers to enhance productivity and contribute to open-source projects.

## Tools Included

### 1. Markdown Link Checker (`link_checker.py`)
Checks for broken links in Markdown files.
- Usage: `python link_checker.py <markdown_file>`

### 2. README Generator (`readme_generator.py`)
Generates a basic README.md template for new projects.
- Usage: `python readme_generator.py <project_name>`

### 3. Code Line Counter (`line_counter.py`)
Counts lines of code in Python files within a directory.  
- **Usage**: `python line_counter.py <directory>`

### 4. JSON Formatter (`json_formatter.py`)
Formats and validates JSON files.  
- **Usage**: `python json_formatter.py <json_file>`

### 5. CSV to JSON Converter (`csv_to_json.py`)
Converts CSV files to JSON.  
- **Usage**: `python csv_to_json.py <csv_file> <json_file>`

### 6. Simple HTTP Server (`simple_server.py`)
Serves files from the current directory.  
- **Usage**: `python simple_server.py [port]`

### 7. File Renamer (`file_renamer.py`)
Renames files in a directory with regex patterns.  
- **Usage**: `python file_renamer.py <directory> <old_pattern> <new_pattern>`

### 8. Git Log Analyzer (`git_analyzer.py`)
Analyzes git log for commit stats.  
- **Usage**: `python git_analyzer.py [repo_path]`

### 9. Password Generator (`password_gen.py`)
Generates secure random passwords.  
- **Usage**: `python password_gen.py [length]`

### 10. Todo List Manager (`todo.py`)
A simple CLI todo list.  
- **Usage**: `python todo.py [add <task> | list | done <index>]`

### 11. Text File Merger (`file_merger.py`)
Merges multiple text files into one.  
- **Usage**: `python file_merger.py <output_file> <input_file1> ...`

### 12. URL Shortener (`url_shortener.py`)
Simple local URL shortener.  
- **Usage**: `python url_shortener.py [shorten <url> | get <short>]`

### 13. Weather Fetcher (`weather.py`)
Fetches weather using wttr.in.  
- **Usage**: `python weather.py <city>`

### 14. Simple Calculator (`calculator.py`)
Evaluates basic math expressions.  
- **Usage**: `python calculator.py <expression>`

### 15. File Finder (`file_finder.py`)
Finds files by name in a directory.  
- **Usage**: `python file_finder.py <directory> <filename>`

### 16. Word Counter (`word_counter.py`)
Counts words in a text file.  
- **Usage**: `python word_counter.py <file>`

### 17. Backup Script (`backup.py`)
Copies files from source to backup directory.  
- **Usage**: `python backup.py <source_dir> <backup_dir>`

### 18. Note Taker (`notes.py`)
Simple note taking CLI.  
- **Usage**: `python notes.py [add <note> | list | clear]`

## Installation

Clone this repo and ensure you have Python 3 installed.

```bash
git clone https://github.com/yourusername/dev-tools.git
cd dev-tools
```
